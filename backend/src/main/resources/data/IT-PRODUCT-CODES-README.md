# IT Product Codes Excel File Format

## 파일명
`it-product-codes.xlsx`

## Excel 파일 형식

### Sheet 1 (기본 시트)

| Column A (HS코드) | Column B (제품명) | Column C (설명) |
|-----------------|---------------|--------------|
| 8471 | 자동자료처리기계 | 컴퓨터 및 주변기기 |
| 8473 | 자동자료처리기계의 부분품 | 컴퓨터 부품 |
| 8517 | 전화기 및 음성/영상 송수신기기 | 통신기기 |
| 8518 | 마이크로폰, 확성기, 헤드폰 | 음향기기 |
| 8521 | 영상 녹화/재생기기 | 영상기기 |
| 8523 | 디스크, 테이프 및 기타 매체 | 저장매체 |
| 8525 | 송신기기 | 방송장비 |
| 8528 | 모니터, 프로젝터, TV 수신기기 | 디스플레이 |
| 8534 | 인쇄회로 | PCB |
| 8541 | 다이오드, 트랜지스터 | 반도체 소자 |
| 8542 | 전자집적회로 | IC, 메모리칩 |
| 9001 | 광섬유, 광학렌즈 | 광학부품 |
| 9013 | 액정디바이스 | LCD |
| 9027 | 분석/측정기기 | 분석장비 |

## 생성 방법

### 1. Excel 파일 직접 생성
1. Microsoft Excel 또는 LibreOffice Calc 실행
2. 위 표의 내용을 입력
3. 첫 번째 행을 헤더로 설정
4. `it-product-codes.xlsx` 이름으로 저장
5. `src/main/resources/data/` 폴더에 저장

### 2. CSV로 생성 후 Excel 변환
```csv
HS코드,제품명,설명
8471,자동자료처리기계,컴퓨터 및 주변기기
8473,자동자료처리기계의 부분품,컴퓨터 부품
8517,전화기 및 음성/영상 송수신기기,통신기기
8518,마이크로폰 확성기 헤드폰,음향기기
8521,영상 녹화/재생기기,영상기기
8523,디스크 테이프 및 기타 매체,저장매체
8525,송신기기,방송장비
8528,모니터 프로젝터 TV 수신기기,디스플레이
8534,인쇄회로,PCB
8541,다이오드 트랜지스터,반도체 소자
8542,전자집적회로,IC 메모리칩
9001,광섬유 광학렌즈,광학부품
9013,액정디바이스,LCD
9027,분석/측정기기,분석장비
```

## 주의사항

1. **헤더 행 필수**: 첫 번째 행은 헤더로 사용됩니다
2. **HS코드 형식**: 숫자 또는 문자열 (2~10자리)
3. **필수 컬럼**: HS코드, 제품명은 필수
4. **설명 컬럼**: 선택사항 (비어있어도 됨)
5. **인코딩**: UTF-8 권장

## 사용 예시

배치 작업 실행 시 이 파일을 읽어 IT 제품 필터링에 사용합니다.

```yaml
# application.yml
batch:
  customs-sync:
    excel-file-path: classpath:data/it-product-codes.xlsx
```
